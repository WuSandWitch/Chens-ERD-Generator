# Chen's ERD generator

This was made for a Database System assignment.

The assignment needed Chen's Notation ERDs, but I could not find a tool that let AI help me draw them properly. Mermaid did not support this, and the other platforms I checked did not really support Chen's Notation either, so I made this.

You can also open the site and copy the prompt inside, then give it to an AI to help with the assignment.

It is a very random project, but it works.
